# Simple Text Display (React)

Simple example that demonstrates a controlled input component in React.

## How to run

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the dev server:

   ```bash
   npm start
   ```

Open http://localhost:3000 in your browser.

## What it demonstrates

- Controlled components (input value set by React state)
- onChange updates state every keystroke
- Live display of typed text
