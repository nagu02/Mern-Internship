# Button Color Changer

Simple React app that changes the page background color when a button is clicked.

## How to run

1. Install dependencies:

```bash
npm install
```

2. Start the dev server:

```bash
npm start
```

This project is a regular React app using `react-scripts` (Create React App style), not Vite.

## What it demonstrates

- State with `useState`
- Event handling with `onClick`
- Dynamic inline styling via `style={{ backgroundColor: color }}`
