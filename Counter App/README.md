# Counter App (React)

Simple demo showing React's `useState` hook with Increase, Decrease, and Reset buttons.

## Setup

1. Install dependencies

   npm install

2. Run dev server

   npm start

3. Open the printed local URL (usually http://localhost:3000)

## What this shows

- `useState` to hold the `count` value
- Buttons that call state-updating functions (increase, decrease, reset)
- React re-renders automatically when state changes

Files of interest: `src/Counter.jsx`, `src/App.jsx`, `src/index.js`, `public/index.html`
